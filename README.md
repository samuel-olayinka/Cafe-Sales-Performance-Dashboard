
# ☕ Café Sales Performance Dashboard (Excel + Power Query)

## 🔎 Project Overview

This project presents a **Café Sales Performance Dashboard** developed in **Microsoft Excel**, with data processed and cleaned using **Power Query**.

The dashboard provides insights into product performance, revenue distribution, branch comparisons, and customer trends for business decision-making.

## 🔄 Data Cleaning & Transformation

The dataset underwent a structured ETL workflow using Power Query:

* Cleaned inconsistent product and cashier entries
* Converted unit price, quantity, and revenue fields to numeric formats
* Replaced null values with appropriate defaults
* Created calculated fields (Revenue, Category grouping)
* Extracted Month for trend analysis
* Verified dataset accuracy using column profiling tools

---

## 📊 Dashboard Features

### ✔ KPI Summary

* Total Revenue Generated
* Quantity Sold
* Average Unit Price
* Average Customer Rating

### ✔ Sales Analysis

* Product Category Revenue Distribution
* Payment Method Breakdown (POS, Transfer, Cash)
* Monthly Sales Trend Visualization
* Branch Performance Comparison (Lagos, Abuja, Port Harcourt)

### ✔ Interactivity

* Cashier-level filtering
* Category-based filtering
* Dynamic visual updates

---

## 📈 Key Insights

* Lagos branch generated the highest sales performance.
* Croissants and Lattes are top revenue-driving products.
* Sales fluctuate monthly, indicating seasonal demand patterns.
* Digital payments contribute significantly to transactions.

---

## 🎯 Project Objective

To showcase how Excel dashboards can support **retail business intelligence**, enabling stakeholders to track performance, identify profitable products, and guide operational strategy.
